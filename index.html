Tutorial Calculadora

A calculadora possui diferentes sistemas que podem ser selecionados. Cada sistema possui diferentes configurações e entradas necessárias. Vamos entender como utilizar a calculadora passo a passo:

Passo 1: Escolha o sistema

No seletor de sistemas, escolha o sistema desejado.

Conforme a seleção do sistema, as entradas e informações na tela serão atualizadas.

Passo 2: Adicione os blocos (somente para alguns sistemas)

Se o sistema escolhido exigir a adição de blocos, você verá um campo "Quantidade de Blocos" abaixo do seletor de sistemas.

Insira a quantidade de blocos que deseja adicionar e clique no botão "Adicionar Blocos".

Passo 3: Preencha as informações necessárias

Após adicionar os blocos (se aplicável), você verá as entradas correspondentes ao sistema escolhido.

Preencha as informações solicitadas, como massas, coeficiente de atrito, ângulo, etc. Certifique-se de inserir os valores corretos de acordo com as unidades especificadas.

Passo 4: Realize o cálculo

Após preencher todas as informações necessárias, clique no botão "Calcular" para obter o resultado.

Observações:

Alguns sistemas podem solicitar se há atrito ou não. Caso seja solicitado, escolha entre "Sim" ou "Não".

Se o sistema possuir atrito, serão exibidos campos adicionais para inserir os coeficientes de atrito estático e dinâmico.
